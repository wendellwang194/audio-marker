# 音频标注切分工具 — 功能说明

> 文件位置：`audio-marker/index.html`，双击即可在浏览器中打开使用  
> 无需安装任何依赖，纯前端离线运行

---

## 一、界面布局

```
┌─────────────────────────────────────────────────┐

├─────────────────────────────────────────────────┤
│  波形可视化区域（可缩放、滚动、点击跳转）         │
├─────────────────────────────────────────────────┤
│  时间轴                                          │
├─────────────────────────────────────────────────┤
│  进度条（可拖拽 + 时间码跳转）                   │
├─────────────────────────────────────────────────┤
│  ▶ 播放控制区  音量  速度  【添加标记】 撤销    │
├──────────────┬──────────────────────────────────┤

```

## 二、顶部操作栏

### 📂 导入音频
- 支持常见音频格式（mp3、wav、m4a、ogg、flac 等）
- 导入后自动解码并在波形区绘制波形
- 快捷键 **I**

### 📊 导入 Excel
- 导入之前导出的切片 Excel 文件（`.xls` / `.xlsx` / `.csv`）
- Excel 格式要求：
  - 第1列：开始时间码（支持 `SS.M` / `MM:SS` / `MM:SS.M` / `HH:MM:SS` / `HH:MM:SS.M`）
  - 第2列：结束时间码
  - 第3列：切片名称
- 导入后自动恢复所有标记点和切片
- **注意：导入 Excel 前必须先导入音频**

### 📊 导出 Excel
- 将当前所有切片导出为 `.xls` 文件
- 包含三列：开始时间 / 结束时间 / 切片名称
- 文件名自动使用音频文件名

### 📦 导出切片
- 将所有切片打包为 ZIP 文件一次性下载
- 每个切片为独立 WAV 文件
- 文件名使用切片名称，默认为 `切片 01`、`切片 02`……
- 快捷键 **E**

### 🗑️ 清空标记
- 清除所有标记点和切片
- 音频保留，可重新标记

### 快捷键提示
- 顶栏右侧显示常用快捷键说明

---

## 三、波形可视化区域

### 波形绘制
- 导入音频后自动绘制深色主题波形
- 开始标记显示为 **绿色竖线**
- 结束标记显示为 **红色竖线**
- 标记线上方显示时间码和名称

### 交互操作
| 操作 | 效果 |
|------|------|
| **点击波形** | 播放头跳转到点击位置 |
| **点击标记线** | 播放头跳转到该标记的时间点 |
| **鼠标滚轮** | 在波形区上下滚动时，若按住 Ctrl 可缩放波形 |

### 缩放与滚动
- **缩放滑块**（波形区下方）：拖动可放大/缩小波形，范围 1x ~ 50x
- **滚动条**（缩放滑块右侧）：当波形超出可视宽度时出现，拖拽可左右滚动查看不同位置
- 缩放时波形宽度变化，滚动条自动适配

---

## 四、时间轴

- 波形区下方显示时间刻度尺
- 根据当前缩放级别自动调整刻度间隔
- 方便快速定位时间点

---

## 五、进度条

### 播放进度
- 紫色渐变进度条实时显示播放进度
- 圆形滑块可拖拽跳转播放位置

### 点击跳转
- 点击进度条任意位置，播放头跳转到对应时间

### 时间码输入
- 进度条左侧显示当前时间 / 总时长
- 中间输入框可手动输入时间码跳转
- 支持格式：`SS.M` / `MM:SS` / `MM:SS.M` / `HH:MM:SS` / `HH:MM:SS.M`
- 输入后按回车或点击 → 按钮确认跳转

### 悬停预览
- 鼠标悬停在进度条上时，显示气泡提示当前时间

---

## 六、播放控制区

### ▶ 播放 / 暂停
- 圆形按钮，点击切换播放 / 暂停状态
- 快捷键 **Space**（空格键）

### 时间显示
- 实时显示当前播放时间 / 总时长
- 格式：`MM:SS.M / MM:SS.M`

### 音量控制 🔊
- 滑块调节音量，范围 0% ~ 200%
- 默认 100%

### 播放速度控制 ⚡
- 滑块调节播放速度，范围 0.5x ~ 2.0x
- 默认 1.0x
- 快捷键 **[** 减速 / **]** 加速

### 【添加开始标记】/【添加结束标记】
- **开始标记**（绿色）：标记切片的起点，快捷键 **[**
- **结束标记**（红色）：标记切片的终点，快捷键 **]**
- 标记会自动按时间排序，并配对生成切片

### ↩️ 撤销
- 撤销最近一次添加/删除的标记
- 快捷键 **Z**（需配合 Ctrl/Cmd）

---

## 七、标记点列表（左侧面板）

> 显示所有已添加的标记点，按时间顺序排列

### 每条标记显示内容
| 元素 | 说明 |
|------|------|
| 类型徽章 | 绿色「开始」/ 红色「结束」/ 黄色「错误」 |
| 时间码 | 可编辑，修改后自动更新波形和切片 |
| 名称 | 开始标记可自定义名称，会同步到切片列表 |
| ↺ 按钮 | 将该标记时间更新为当前播放位置 |
| 跳转按钮 | 点击后播放头跳转到该标记时间 |
| 删除按钮 | 删除该标记（×） |

### 时间码编辑
- 点击时间码进入编辑状态
- 支持任意时间格式输入
- 回车或失焦后自动应用
- 非法值会显示红色边框提示

### 名称编辑
- 开始标记的名称会影响对应切片的显示名称
- 在标记列表或切片列表中编辑均可同步

---

## 八、切片列表（右侧面板）

> 显示所有由「开始-结束」标记配对生成的切片

### 每条切片显示内容
| 元素 | 说明 |
|------|------|
| 序号 | 紫色 `#1` `#2` … 标签 |
| 切片名称 | 可编辑，默认为「切片 1」「切片 2」… |
| 开始时间 | 可编辑，修改后自动更新 |
| 结束时间 | 可编辑，修改后自动更新 |
| ↺ (开始) | 将该切片的开始时间更新为当前播放位置 |
| ↺ (结束) | 将该切片的结束时间更新为当前播放位置 |
| ▶ 播放 | 播放该切片（播放完后自动停止） |
| ◼ 停止 | 停止播放（播放中显示） |

### 切片播放
- 点击「▶ 播放」按钮，只播放该切片对应的时间段
- 播放完毕后自动停止，播放头回到切片起点
- 播放中按钮变为「◼ 停止」，点击可手动停止

### 时间码编辑
- 开始/结束时间均可直接编辑
- 回车或失焦后自动应用
- 修改后波形标记线和切片列表同步更新

---

## 九、导出功能详解

### 导出 Excel
- 点击「📊 导出Excel」按钮
- 自动下载 `.xls` 文件（兼容 Excel 和 WPS）
- 文件内容：

| 开始时间 | 结束时间 | 切片名称 |
|---------|---------|---------|
| 00:03.0 | 00:15.5 | 切片 1 |
| 00:20.0 | 00:45.2 | 切片 2 |

### 导出切片（ZIP）
- 点击「📦 导出切片」按钮
- 自动打包所有切片为 ZIP 文件下载
- 每个切片为独立 WAV 文件
- 音频参数与原文件一致（采样率、声道数相同）

---

## 十、导入功能详解

### 导入 Excel 恢复切片
1. 先点击「📂 导入音频」加载音频文件
2. 再点击「📊 导入Excel」选择之前导出的 `.xls` 文件
3. 系统自动解析并恢复所有标记点和切片
4. 波形上会显示对应的标记线

---

## 十一、完整快捷键列表

| 快捷键 | 功能 |
|--------|------|
| **Space** | 播放 / 暂停 |
| **I** | 选择并导入音频文件 |
| **[** | 添加开始标记（绿色） |
| **]** | 添加结束标记（红色） |
| **E** | 导出切片 ZIP |
| **Z** | 撤销最近一次标记操作 |

---

## 十二、注意事项

1. **浏览器兼容**：推荐使用 Chrome / Edge / Safari 最新版
2. **大文件**：音频文件过大时解码需要等待，请耐心等待进度条消失
3. **导入 Excel 前必须先导入音频**，否则会弹出提示
4. **时间码格式**：导入 Excel 时，时间码支持多种格式，系统会自动识别
5. **标记配对规则**：按时间顺序，第1个开始标记与第1个结束标记配对，以此类推；若开始/结束数量不等，会显示黄色警告提示
